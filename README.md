# 🍕 SmartPizza AI – Intelligent Food Ordering & Delivery System

## 📌 Project Description
SmartPizza AI is a full-stack web application designed to automate and optimize pizza ordering and delivery.

## 🚀 Features
- User Login & Registration
- Pizza Ordering
- Cart & Coupon Management
- Payment Integration
- Delivery Tracking
- AI Recommendations
- Admin Dashboard

## 🏗️ Tech Stack
- React.js
- Spring Boot
- MySQL
- JWT Authentication

## 📂 Project Structure
Backend & Frontend separation using layered architecture.

## ⚙️ Setup
Backend:
    mvn spring-boot:run
Frontend:
    npm install
    npm start
